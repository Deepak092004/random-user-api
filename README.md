# Random User API Project

This is a simple Python project that fetches random user data from a public API and displays basic user information like username and location.

## Features
- Fetches random user data using REST API
- Uses `requests` library
- Clean and beginner-friendly Python code
- Proper error handling

## Technologies Used
- Python
- Requests library
- REST API

## API Used
https://api.freeapi.app/api/v1/public/randomusers/user/random
